* Create a new branch named 'feature'
* Switch to your 'feature' branch
* Make a change to a file, and commit it
* Create and switch to a new branch based off master named 'release'
* Make a change to a different file, and commit it
* Switch back to the master branch
* Make some changes, then stash them
* Merge the 'feature' branch into 'release'
* Merge the 'release' branch into master
* Apply your stashed changes and commit them to master
